# For this project, install the workload:

.NET desktop development
That workload includes the Windows Forms and .NET Framework tools needed to build and run this application.

If you want a minimal setup, also make sure these components are selected inside that workload:

.NET 4.8 SDK / targeting pack
Windows 10/11 SDK
C# development tools

# To open this app on Windows, use Visual Studio:

Open ThreatScanner.sln or click file ThreatScanner.csproj and it will go in Visual Studio.
Restore NuGet packages if prompted.
Make sure the project named ThreatScanner is the startup project.
Press F5 or click Start Debugging.
This project is a Windows Forms application, and its entry point is Program.cs.

If you do not have Visual Studio installed, install Visual Studio 2022 with the .NET desktop development workload first.
